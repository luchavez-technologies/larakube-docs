# 📁 Enterprise Portfolio

A professional, high-traffic portfolio or corporate site that's easy to manage.

### 🏗 The Blueprint
This recipe focuses on simplicity, speed, and clean content management.

-   **Command**: `larakube new portfolio --statamic --nginx --sqlite`
-   **CMS**: [Statamic](https://statamic.dev) offers a clean, flat-file CMS experience that's perfect for version-controlled content.
-   **Server**: [Nginx](https://laravel.com/docs/13.x/deployment#nginx) provides a traditional and widely-trusted web server foundation.
-   **Stability**: Using **SQLite** ensures your portfolio remains ultra-lightweight and fast to scaffold.

---
