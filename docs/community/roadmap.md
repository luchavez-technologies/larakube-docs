# 🗺️ Project Roadmap

LaraKube is an ecosystem built on three pillars: the **CLI** (The Engine), the **Console** (The Brain), and the **Docs** (The Map). This roadmap tracks our vision for scaling professional Laravel orchestration.

---

## 🟢 Phase 1: Foundations (Current)
*Focus: Stability, Local Parity, and Core Blueprints.*

- [x] **CLI**: Industrial-strength orchestration engine with zero-host dependencies.
- [x] **CLI**: Automated "Single-Node Hero" VPS graduation path ($6 baseline).
- [x] **CLI**: Dual-MCP architecture for AI-native management.
- [x] **Console**: High-fidelity Kubernetes Control Plane (Filament v5).
- [x] **Docs**: Linear learning path with Mermaid visual schematics.

---

## 🟡 Phase 2: Professional Expansion (Q2 2026)
*Focus: Security, Monitoring, and Developer Experience.*

- [ ] **Console**: Integrated **LaraKube Chat** (rebuilt for the stateful Control Plane).
- [ ] **CLI**: One-click **Multi-Cloud Provisioning** (DOKS, GKE, AWS EKS).
- [ ] **Ecosystem**: Support for **Custom Storage Providers** (DigitalOcean Spaces, AWS S3).
- [ ] **Monitoring**: Real-time Prometheus/Grafana integration inside the Console.
- [ ] **Security**: Automated **Security Audits** for production manifests.

---

## 🔵 Phase 3: Enterprise Scale (2027+)
*Focus: Automation, Redundancy, and Growth.*

- [ ] **CLI**: **Autoscaling Logic** for production workloads (HPA).
- [ ] **Console**: Multi-cluster fleet management from a single dashboard.
- [ ] **Ecosystem**: Marketplace for specialized "Masterpiece Blueprints."
- [ ] **Security**: RBAC management for large teams directly via the Console.

---
